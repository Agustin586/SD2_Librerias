Descripción detallada

Libreria para manipulacion del modulo can mcp2515.

Autor
Zuliani, Agustin.
Fecha
03/08/24
Nota
Para utilizar dicha libreria debera incluirse de antemano la libreria correspondiente del modulo spi para trabajar con la kl46z. En adicion esa libreria debera corresponder a la que permite el uso de freertos si fuera el caso. Para nuestro caso utilizamos spi.h y spi.c para trabajar.
